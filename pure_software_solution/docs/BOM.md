# AdMute - Pure Software Bill of Materials (BOM)

Because this architecture relies on network commands and ambient listening rather than complex audio routing, the hardware requirements are extremely minimal. 

You **do not** need breadboards, resistors, LEDs, push buttons, or external Bluetooth dongles.

## Required Hardware

| Item | Description | Estimated Cost |
| :--- | :--- | :--- |
| **Raspberry Pi** | A Raspberry Pi 3, 4, or 5 (with built-in Wi-Fi). A Pi Zero 2 W will also work. | $35 - $60 |
| **USB Microphone** | A basic plug-and-play USB microphone. The Pi just needs to listen to the ambient room audio coming from your TV speaker. | $5 - $15 |
| **MicroSD Card** | At least 16GB. Must be high quality (e.g., SanDisk Ultra or Samsung EVO) to run Raspberry Pi OS reliably. | $8 |
| **Power Supply** | Official Raspberry Pi USB-C (or Micro-USB for older models) power supply. | $8 - $10 |

## Optional Hardware

| Item | Description | Estimated Cost |
| :--- | :--- | :--- |
| **Protective Case** | A standard case to protect the Raspberry Pi from dust and short circuits while it sits near your TV. | $10 |

---
**Total Estimated Hardware Cost:** ~$60 - $100 *(Significantly cheaper if you already own a Raspberry Pi)*
